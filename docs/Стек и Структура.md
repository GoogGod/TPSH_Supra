# Стек и структура — Supra

---

## Контекст проекта

**Supra** — автоматизированная система планирования трудовых ресурсов с ML-прогнозом нагрузки и генерацией расписаний.

### Роли

| Роль | Код в системе | Возможности |
|------|---------------|-------------|
| Сотрудник (стажёр) | `employee_noob` | ЛК, просмотр расписания, claim слотов, уведомления |
| Сотрудник (опытный) | `employee_pro` | То же + приоритет при назначении (на будущее) |
| Менеджер | `manager` | + управление расписанием, запуск ML, назначение сотрудников, загрузка данных |
| Администратор | `admin` | + управление пользователями, объектами, метрики модели |

### Текущий scope

- Backend API: auth, users, shifts, forecasting, notifications
- ML-пайплайн: обработка raw-данных → обучение модели → прогноз → генерация расписания
- Frontend: Vue 3 SPA
